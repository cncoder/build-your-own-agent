Agent 的心跳：Always-On 运行时

读者学到什么
Heartbeat 让 agent 从「被动响应」变为「主动巡检」。实现只需一个 while True + asyncio.sleep(interval)——agent 每隔 N 秒唤醒，检查任务队列、触发 cron。漏跳监控通过外部看门狗实现：若 N+buffer 秒内未收到心跳 ping，则告警或自动重启。
