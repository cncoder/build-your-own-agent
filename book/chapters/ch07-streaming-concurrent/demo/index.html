读者学到什么
流式输出的本质是 HTTP 长连接中 Content-Type: text/event-stream——服务器持续 push「data: {...}\n\n」事件，浏览器 EventSource API 监听并逐片追加。用户感知的等待时间从「全部生成完毕」缩短为「第一个 token」到达，通常从 3-8 秒缩短到 150-400ms。
