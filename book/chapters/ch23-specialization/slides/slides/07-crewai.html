Crew 自主协作
一次性任务，多角色协作完成
# 四层 agent 协作（对应 TradingAgents） analyst = Agent(role="市场分析师", goal="识别交易信号", tools=[price, indicator]) researcher = Agent(role="基本面研究员", goal="收集新闻情绪", tools=[news, onchain]) trader = Agent(role="交易执行员", goal="综合信号执行策略", tools=[order, pos], llm="us.anthropic.claude-opus-4-7") crew = Crew( agents=[analyst, researcher, trader, risk], tasks=[analysis_task, research_task, trade_task, risk_task], process=Process.sequential ) result = crew.kickoff()
Crew 适用：一次性分析报告、有自然专业分工的任务、需要 agent 互相传递上下文
Flow 事件驱动
持续运行 pipeline，状态机 + 动态分支
# 量化交易持续监控 Flow class TradingFlow(Flow[TradingState]): @start() def scan_market(self): signal = scanner.scan() self.state.signal = signal return signal @listen(scan_market) @router(lambda s: "high" if s.conf > 0.8 else "low") def route_signal(self, s): pass @listen("high") def check_risk(self, s): approved = risk.check(s) return "approved" if approved else "rejected" @listen("approved") def execute_trade(self): trade.execute(self.state.signal) TradingFlow().kickoff()
Flow 适用：持续运行监控、需要动态分支（高置信度/低置信度走不同路径）、长期 pipeline（播客生产/每日任务）
Crew = 项目制（一次性，协作完成一个任务）
Flow = 产品制（持续运行，事件驱动，状态机）
