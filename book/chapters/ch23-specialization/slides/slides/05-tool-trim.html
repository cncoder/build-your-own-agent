为什么裁剪能减少幻觉？
工具越少 → 选错概率越低
LLM 扫描所有工具来决策，30 个工具比 7 个工具选错的概率高 4 倍以上
隐式安全边界
TradingBot 不可能"不小心"去读取配置文件——它根本没有 read_file 工具
更短的 context
工具定义少了 → 系统 prompt 短了 → 更多 token 给推理
