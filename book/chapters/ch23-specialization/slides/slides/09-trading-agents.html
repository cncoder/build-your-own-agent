TradingAgents · 角色扮演多 agent 通用套路
四层 agent 层级：分析师 → 研究员 → 交易员 → 风控
LAYER 1
分析师
专注技术指标分析
不看新闻，不看情绪
工具：get_price
get_indicators
RSI / MACD / BB
输出：
RSI=42 (中性)
MACD 金叉
BB 中轨上方
多头置信度 0.72
→
LAYER 2
研究员
基本面 + 新闻情绪
补充技术面的盲区
工具：collect_news
onchain_data
Fear & Greed Index
输出：
新闻情绪偏多
大户持仓 +3%
F&G = 58
基本面支持多头
→
LAYER 3
交易员
综合 L1+L2 制定策略
Opus 级别推理
工具：place_order
close_position
get_positions
输出：
开多 BTC 0.1 张
入场 $103,200
止损 $101,648
止盈 $106,296
→
LAYER 4
风控官
最后一道门
代码级硬编码，不可绕过
工具：validate_risk
check_exposure
circuit_breaker
审核：
敞口 $103 ≤ $206 ✅
今日亏损 $0 ✅
熔断 OFF ✅
→ 批准执行
专注减少幻觉分析师只看技术，不会被新闻噪音干扰，决策更准
并行提速L1 分析师 + L2 研究员可以并行执行，总时间减半
职责可替换换策略只改 L3 交易员，L1/L2 不动；风控永远是 L4
对应 hermesbot[案例 19.1] 的 Python 实现，用 Strands SDK 而非 LangGraph
