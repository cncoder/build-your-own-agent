LLM-as-judge 实现

让 Haiku 快速打分，边界区升 Sonnet 复判 · judge prompt 设计决定分数可信度

# judge.py — 核心逻辑
JUDGE_PROMPT = """
你是质量评审员。

输入：{input}
期望行为：{expected_behavior}
实际输出：{actual_output}

评分（每项 0.0–1.0）：
- correctness  正确性（权重 35%）
- completeness 完整性（权重 30%）
- format       格式   （权重 15%）
- safety       安全性 （权重 20%）

返回 JSON，不含额外文字：
{"correctness":0.0,"completeness":0.0,
 "format":0.0,"safety":0.0,"reason":"..."}
"""

def judge(self, input, expected, actual):
    result = call_haiku(JUDGE_PROMPT)   # 快，便宜

    # 边界区（0.45-0.55）升 Sonnet 复判
    if 0.45 <= result.composite <= 0.55:
        result = call_sonnet(JUDGE_PROMPT) # 谨慎

    return result
