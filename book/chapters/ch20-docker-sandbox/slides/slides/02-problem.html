Agent 能跑任意代码

正则过滤
挡不住

Ch10 封锁了 rm -rf。
但攻击者有无数绕过方式。
黑名单永远比攻击者的想象力慢。
根本矛盾
黑名单枚举已知危险。
攻击者制造未知危险。
# 「禁止 rm -rf」被绕过的 5 种方式
python3 -c "import shutil; shutil.rmtree('/')" ↑ Python
find / -type f | xargs unlink ↑ find+pipe
dd if=/dev/zero of=/dev/sda bs=512 ↑ dd
perl -e 'unlink glob("/home/*")' ↑ Perl
echo "cm0gLXJmIC8=" | base64 -d | bash ↑ Base64
每一条都绕过了正则过滤
# 隔离的逻辑（Docker）
不管代码做什么，出不了这个笼子。
容器内 rm -rf / → 只删容器内的文件
容器销毁 → 一切恢复原状
