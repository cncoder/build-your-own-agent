Ch 14 · 执行安全

三层安全门：沙盒 · 审批 · 凭据隔离

通用 agent 能执行任意代码，这意味着它也能误删文件、泄露密钥。三层安全机制从不同维度守卫执行边界，让「强大」和「可控」同时成立。

三层防御
🔐 Approval Gate 待触发
高风险操作（删除/部署/外发）需人工审批后才能继续执行。
📦 Sandbox 待触发
代码在隔离容器中执行，无网络、无宿主文件系统访问，token 不可达。
🗝️ Vault（凭据隔离） 待触发
API Key / Secret 不注入 agent context，仅在执行层按需取用，沙盒内永不可读原文。
