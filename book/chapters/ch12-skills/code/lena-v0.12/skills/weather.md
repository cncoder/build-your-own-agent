查询 **$ARGUMENTS** 的当前天气，按以下 SOP 处理：

## 步骤

1. **确认城市名**：如果 $ARGUMENTS 是缩写或别名（如"魔都"→上海），先转换为标准地名
2. **调用工具**：使用 get_weather 工具获取天气数据
3. **格式化输出**：
   - 当前温度（摄氏度）和体感温度
   - 天气描述（晴/多云/雨/雪等）
   - 湿度和风速
   - 未来 24 小时趋势（用 ▲ 表示升温，▼ 表示降温）
4. **出行建议**：根据天气给出一句话出行建议（是否需要带伞、防晒等）

## 错误处理

- API 返回错误：回复"$ARGUMENTS 的天气服务暂时不可用，请稍后重试"，不暴露技术细节
- 城市名无法识别：询问用户是否输入正确，提供几个可能的相似城市名
- 网络超时：提示用户检查网络，建议重试

## 输出格式示例

```
🌤 上海天气（2026-05-05 14:00）
温度：22°C（体感 20°C）
天气：多云转晴
湿度：65% | 风速：3级东北风

未来 24h：▲ 明日最高 26°C，午后有阵雨可能

💡 建议随身带伞，下午出门注意阵雨。
```
