## F. 大厂 JD 共性分析（R17 摘要）

> 本节非具体文献引用，为 2026 上半年大厂 AI Agent 工程师招聘要求的聚合分析。
> 数据窗口：2025-11-01 至 2026-05-07；样本：35 条 JD（Anthropic 7 / Amazon-AWS 15 / Google DeepMind 6 / 中小公司 7）。

**六大高频技能要求（按 JD 提及率降序）**：

1. **Python + LLM API 应用**（94%）：直接调用 Anthropic / OpenAI / AWS Bedrock API，不依赖框架
2. **Agent 架构设计**（75%）：ReAct loop、Orchestrator-Worker、多 Agent 编排
3. **高级提示工程**（81%）：系统提示设计、思维链、上下文窗口管理
4. **工具使用与 MCP**（62%）：function calling、MCP server/client、工具权限控制
5. **安全与可控性**（59%）：prompt injection 防御、沙箱隔离、最小权限原则
6. **可观测性与评估**（56%）：OpenTelemetry GenAI span、LLM judge、evals 体系建立

**核心洞察**：2026 年大厂已不再关注"会不会用 LangChain"，而是关注"能否从底层设计 agent 架构并上生产"。Evals 能力从边缘走向核心（同比提升 23 个百分点）。

**参考数据来源**：Anthropic Greenhouse、Amazon Jobs、Google DeepMind Greenhouse、LinkedIn、levels.fyi（均经 R17 研究 agent 独立核验）
