# D8 · 信息密度判官

## 任务

逐段打标签：

- **新信息**：这段推进了新概念/机制/代码/决策
- **复述**：上一段已讲过，换种说法重复
- **灌水**：连接词、口号、重复强调，无实质内容
- **堆砌**：塞多个不相关术语，像为凑关键词

## 算分

```
A = 新信息段数
B = 复述
C = 灌水
D = 堆砌
total = A + B + C + D

score = A / total × 10
```

附加：堆砌段 ≥ 3 个，再扣 1.5 分。

## 反作弊判据

- 一段串多个术语但每个不展开 → 堆砌
- "这就是 X 落地时最实用的指引" → 灌水
- 段中突然塞入本段主题无关的术语 → 堆砌
- 末尾集中回顾已讲内容 → 复述

## 输入

`{target_path}`

## 输出

```json
{
  "score": 0.0 到 10.0,
  "detail": "A=X B=Y C=Z D=W 总X段；指出最严重的堆砌/灌水段落（引句前20字+判语）"
}
```

detail 500 字以内。
